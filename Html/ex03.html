- Faça uma breve pesquisa sobre as seguintes linguagens -


HTML/CSS

Em teoria, essas duas são linguagens separadas. Porém CSS é tão dependente de HTML que podemos considerá-las juntas. HTML é a principal linguagem que estrutura as páginas web. É ela que define exatamente quais objetos terão na página, através de marcações. Já a CSS faz a estilização visual desses objetos. Com ela, é possível fazer coisas mais básicas (mudar a posição, a orientação, cor e textura de um objeto) ou tarefas mais elaboradas, como animações.

PHP

PHP é uma linguagem muito importante para qualquer programador. Ela é utilizada principalmente no desenvolvimento web, agindo como um “esqueleto” para as páginas, fazendo trocas de informações e intermediando as requisições entre páginas e bancos de dados. Isso possibilita a interação do usuário com botões, links e formulários, tornando os sites mais dinâmicos. Ela é bastante simples para os iniciantes, mas ao mesmo tempo conta com diversas funcionalidades avançadas para programadores experientes e para os desenvolvedores full-stack.

JavaScript

Hoje em dia, JavaScript é uma linguagem de programação que está presente em quase todas as páginas web. Ela tem diversas aplicações muito interessantes, principalmente em termos de interação com o usuário e dinâmica da página. Ela é fundamental para desenvolvedores web, e dialoga diretamente com outras linguagens (principalmente HTML e CSS), atuando na chamada front-end (o lado do cliente). JavaScript torna possível o preenchimento de formulários, botões funcionais e atualização de dados em tempo real, por exemplo.

Python

O Python tem destaque entre as principais linguagens de programação. Essa linguagem é muito versátil e possui diversas utilizações, como gerenciamento de Big Data, processamento de dados, inteligência artificial, entre muitas outras. E ela faz tudo isso de uma maneira muito simples e descomplicada, com uma sintaxe fácil de entender. Além disso é a linguagem que conquistou os corações dos desenvolvedores, e sua comunidade é uma das mais dispostas a resolver problemas. Por conta de tudo isso, programadores Python são muito buscados pelo mercado de trabalho. Prova disso é que grandes empresas como Google e Microsoft utilizam a linguagem em suas aplicações. Segundo uma pesquisa do site Coding Dojo, Python é a terceira linguagem de programação mais procurada pelo mercado.
